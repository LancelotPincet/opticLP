**opticLP**

----

**A library that calculates simple calculations for optical alignments.**

.. code-block:: text

   Author        : Lancelot PINCET
   Library       : opticLP
   GitHub        : https://github.com/LancelotPincet/opticLP
   HTTPS         : https://github.com/LancelotPincet/opticLP.git
   SSH           : git@github.com:LancelotPincet/opticLP.git
   PyPI          : https://pypi.org/project/opticLP
   Docs          : https://opticLP.readthedocs.io

**opticLP** is a library written by **Lancelot PINCET**.
The source code can be consulted on its `GitHub repository <https://github.com/LancelotPincet/opticLP>`_.
However for simply using the features, you can install it in your Python environment using `pip <https://pypi.org/project/opticLP>`_.
And for information, do not hesitate to consult the `documentation <https://opticLP.readthedocs.io>`_.

1. New to **opticLP**? Check out the :doc:`Beginner Guide <beginner_guide>` to get up and running.
2. Want a deeper understanding of key concepts in **opticLP**? Dive into the :doc:`User Guide <user_guide>` to get an in-depth understanding of the tool.
3. Have a doubt on a syntax in **opticLP**? Consult the :doc:`Reference Guide <reference_guide>` to have detailed information on how to call the objects.
